# KALK
Vi skal lage kalkulator (⌐■_■)

Vi utfordrer hverandre på å lage en kalkulator innen ettermiddagen lørdag 27.03 \n
Kalkulatoren skal inneholde addering, subtrahering, gange, dele. \n
Du skal kunnne angre på inntasting. \n
Du skal kunne "gjøre ting" med resultatet av utregningen.
Kalkulatoren skal vise en logg over handlinger som er gjort
Det skal ikke være noe begrensning på hvor mange beregninger som blir gjort.
